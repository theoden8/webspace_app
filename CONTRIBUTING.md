# Contributing to WebSpace

Thanks for your interest in WebSpace! The best ways to contribute are by reporting bugs and suggesting features.

## Reporting Bugs

Open an issue at [github.com/theoden8/webspace_app/issues](https://github.com/theoden8/webspace_app/issues) with:

- What happened vs. what you expected
- Steps to reproduce
- Device, OS version, and app version
- App logs if available (App Settings > App Logs > Copy)

## Suggesting Features

Open an issue with the **feature request** label. Describe the problem you're trying to solve and how you'd like it to work.

## Code Contributions

If you'd like to submit a PR, please open an issue first to discuss the change. This helps avoid duplicated effort and ensures the change fits the project direction.
